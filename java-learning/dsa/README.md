## 💡 Notes & Insights

 * **RomanToInteger:** Practiced handling edge cases where subtraction is required (e.g., IV, IX).
 * **CustomArrayList:** Implemented this to truly understand how resizing and index management work behind the scenes in Java's ArrayList.
