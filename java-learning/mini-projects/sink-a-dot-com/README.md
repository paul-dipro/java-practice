# Sink a Dot Com

A small Java console game based on the **Head First Java** early OOP practice.

## What this project does

- Creates one dot game object
- Stores 3 target locations using `ArrayList<String>`
- Takes user input from the console
- Checks whether the guess is a:
  - `miss`
  - `hit`
  - `kill`
- Counts total guesses until the target is destroyed
