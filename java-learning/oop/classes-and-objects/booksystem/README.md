# Book System — Classes and Objects

A simple book management example demonstrating core class and object concepts in Java.

## Files

| File | Role |
|------|------|
| `Book.java` | Blueprint — fields, constructor, instance method, static method |
| `Main.java` | Entry point — creates objects, calls methods, prints output |

## What it demonstrates

- Object creation from a class (`new Book(...)`)
- Private fields with a getter (`getPages()`)
- Instance method (`display()`)
- Static method that operates on an array of objects (`calculateTotalPages()`)
- Null-safe iteration over a partially filled array
